#### Avoiding Social Engineering
Manipulative tactics used by attackers to trick people
Shoulder surfing
- peek over one's shoulder to get info
Eavesdropping
- Tries to listen to private convos
Piggybacking/Tailgating
- following someone with access
Dumpster diving
- scours through trash can to get info
Operational security - OPSEC
- stresses data protection against social engineers for business aspects such as routines, project details and internal procedures.

#### Creating a culture of security
Organizational change management - OCM
- Recognizing the human role in security, ensuring staff engagement and policy adherence.
- Should start at top positions
- Shared responsibility
1. Development phase
	1. Create comprehensive policies
	2. Educate employees about threats
	3. Implement clear guidelines for handling sensitive data
	4. Build a foundation for employee empowerment
2. Execution phase
	1. Training updates
	2. Simulated cyberattacks
	3. Consistent Communication
3. Reporting and Monitoring
	1. Initial monitoring after the rollout of a security program
	2. Recurring check-ins to maintain program integrity
	3. Educate employees on reporting suspicious activities
	4. Encourage an environment where reporting is valued
	5. Create feedback loops to use insights for future actions
